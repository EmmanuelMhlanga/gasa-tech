## 🛠️ Technical Stack & Architecture
This platform is built using a "Performance-First" philosophy:

* **Frontend:** Semantic HTML5, Modern CSS3 (Custom Properties), and Vanilla JavaScript (ES6+).
* **Performance:** Implemented **Service Workers (PWA)** for offline capabilities and instant cache-loading.
* **SEO:** Fully optimized `sitemap.xml` and `robots.txt` tailored for Google Search Console and local Randburg/Johannesburg indexing.
* **Design:** A custom "System Architecture" UI framework focusing on dark-mode aesthetics and high-fidelity micro-interactions.
